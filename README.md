# ZeroFounder

ZeroFounder is an AI co-founder companion that helps solo builders turn rough ideas into launch-ready plans with persistent memory on 0G.

## What it does
ZeroFounder interviews the user, shapes the idea into a product brief, generates an MVP scope, and saves project context so the agent can resume later.

## Key features
- Idea-to-plan workflow.
- AI-generated product brief.
- MVP scoping and next-step suggestions.
- Persistent memory across sessions.
- Built for the Zero Cup on 0G.

## Why it matters
Builders often lose context as ideas grow. ZeroFounder keeps the project state, notes, and decisions available so the user can keep moving without starting over.

## Getting started
1. Clone the repository.
2. Install dependencies.
3. Run the app locally.
4. Enter a rough idea and start the interview flow.

## 0G usage
ZeroFounder uses 0G for persistent project memory and AI-native workflow support.
